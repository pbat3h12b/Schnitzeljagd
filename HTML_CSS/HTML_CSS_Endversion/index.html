Das Spiel

Hallo Leute,
wir sind die PBA3H12B und die PBT3H12B vom b.i.b. International College in Paderborn. Im vierten Semester von unserer Ausbildung war die Aufgabe das sogenannte Vorprojekt. In diesem Projekt arbeitet die gesamte Klasse an einem Projekt, welches von einem Lehrer vorgegeben wird. In diesem Fall ging es um Geocaching. Das Ziel unserer App ist es, dass neue Schüler mit der APP die Umgebung schneller kennenlernen. An verschiedenen Orten an der Schule gibt es Caches die der Schüler finden muss, danach kann er Spiele spielen und sich auf die Suche zum nächsten Cache geben.
